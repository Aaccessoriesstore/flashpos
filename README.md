# ⚡ FlashPOS — نظام الكاشير الذكي المتكامل

نظام كاشير احترافي مبني بـ React + TypeScript + Supabase + Vercel

## المميزات
- 🛒 نقطة بيع متكاملة مع باركود scanner
- 📦 إدارة مخزون ذكية
- 👥 إدارة عملاء وولاء
- 📊 تقارير وتحليلات
- ⚙️ إدارة مستخدمين وورديات
- 💰 دعم كامل للضريبة المصرية 14%

## Tech Stack
- React 18 + TypeScript + Vite
- Supabase (PostgreSQL + RLS)
- Zustand (State Management)
- Recharts (Charts)
- Vercel (Hosting)
